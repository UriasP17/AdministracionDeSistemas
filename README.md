# AdministracionDeSistemas
Administración de Sistemas - Configuración de Red Multi-SO y Automatización de Diagnóstico
Este repositorio contiene la configuracion de red y scripts de diagnostico automatizado para un entorno multi-sistema (Fedora y Windows)

##Tecnologias Utilizadas
--Sistemas operativos: Fedora Workstation, Windows Server 2019, Ubuntu 22.04
--Herramientas de Virtualización: Oracle VM VirtualBox.
